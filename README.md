###### DOWNLOAD FILE
```
wget https://raw.githubusercontent.com/EdyDevz/vnstatmonitor/main/crot && mv /root/crot /usr/bin/ && chmod +x /usr/bin/crot
```

```
nano auth
```

for test tools type `crot` in the terminal
